# Excel to mongo

It's a project that focus in parse a XLSX and store data in mongodb

## status

- [X] Process xlsx
- [X] Store in mongodb
- [X] Api to store xlsx
- [X] FronEnd to Save xlsx
- [X] Api to search data
- [X] Frontend to search data

## Quickstart

### Requirements

- [ ] Docker (optional)
- [ ] Python3.8+
- [ ] Nodejs16+
- [ ] Mongodb

### Running API

#### Setup

```bash
cd backend/src
pip install -r requirements.txt
```

#### Running backend application

```bash
python main.py
```

### Running Frontend

#### Setup application

```bash
cd frontend
npm i
```

#### Running frontend application

```bash
npm run dev
```
